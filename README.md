# Birthday Page

A simple birthday greeting page built with HTML and CSS.

## Live Demo

Once GitHub Pages is enabled for this repository, the site should be available at:

`https://manjuarya16.github.io/manju`

## File

- `index.html`: the main page for the birthday greeting.

## GitHub Pages Setup

1. Open the repository on GitHub: `https://github.com/manjuarya16/manju`
2. Go to `Settings` → `Pages`.
3. Under `Source`, select `main` branch and folder `root` (`/`).
4. Save the settings.

GitHub will publish the site automatically after a few moments.

## Notes

- The page is already pushed to the `main` branch.
- If GitHub Pages does not appear immediately, wait a few minutes and refresh the `Pages` settings page.
